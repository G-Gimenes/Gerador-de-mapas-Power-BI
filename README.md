# Gerador-de-mapas-Power-BI
Script em Python que baixa a malha municipal do IBGE (2022), filtra apenas os municípios de Mato Grosso, renomeia colunas e gera um arquivo TopoJSON. O resultado é salvo em disco para uso em visualizações como Power BI, permitindo mapas interativos dos limites municipais do estado. ✔
